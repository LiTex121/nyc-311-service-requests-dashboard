# 311 Service Request Data 📊
---
## Project Overview 🔷

This is a learning project and the main idea of it is to simply clean and prepare NYC **311 Service Request data** for analysis.

#### Checkbox
- ✅ Creates a new clean .CSV file for our dashboard
- ❌ Working on making a Dashboard (Looking for a library to use) / Might also make more than 1 dashboard.


---
## About The Dataset 

**Data Name:** 311-Service-Requests-from-2020-to-Present
**Data Used:** https://data.cityofnewyork.us/Social-Services/311-Service-Requests-from-2020-to-Present/erm2-nwe9/about_data


---

## Questions This Dataset Answers ⁉️
1. Which boroughs had the most requests?
2. What were the most common complaint types?
3. Which agencies handled the most requests?
4. and more.

## Tools used 🛠️

- Python (Main coding lang)

- Jupyter Notebook
- Virtual Environment (Venv)
- NYC Open Data (311 dataset)

## Project Setup 💻
### Making a Virtual Environment for our Project 

Run the following command on console / Hotkey **Ctrl + Shift `** if using VS Code

```python3 -m venv venv```

### Virtual Environment Activation

#### Windows Only 

```venv\Scripts\activate.bat```

#### MacOS Only 

```source venv/bin/activate```                                                                                                                                                     

## Downloading All Required Dependencies ⤵️
Run the following in the console to automatically download all dependencies:

```pip3 install -r requirements.txt```
